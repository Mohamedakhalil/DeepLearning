Problem Statement

The focus of our problem is to colorize image and generate as realistic colorful images as possible given an image with the lightness channel , grey-scale, only. This has many applications like better representing historic photos making them more digestable to the viewer or for surveillance purposes where it can provide vivid images of a scene.

Dataset

The dataset used for training was the Imagenet dataset which is designed for different deep learning and computer vision applications. Other datasets used in preliminary stages was the 5k test dataset by imagenet.

State of the art

After studying multiple models available in the literture, The state of the art of our problem was found to be the model Colorization Transformer by Kumar et. al. as it scored 42.3% fooling rate.



Orignial Model from Literature

The model we chose as our baseline was the colorful image colorization . The model relies on a feature extractor using CNNs similar to the traditional VGG 16 network but with fewer layers. The extracted feature semantics are later passed to a softmax layer which produces probability distribution of the colors in the form of a_b channels which will be later concatenated with L channel (grey-scale image)



Proposed Updates

This model was chosen in particular as it was the best candidate in terms of possible updates which will be listed below.

Update #1: Introduced a new evaluation metric

A new evaluation metric was used to ensure objective evaluation for all tested images instead of the traditional turing test which is subjective by nature and Pixel by Pixel evaluation metric which can harshly penalize plausible images if they didn't match the original image's colors. This was done using VGG16 where the output and ground truth images were passed to the network and had their classification results compared



Update #2: Apply axial transformation

Axial transformation was used to extract the high level features using an multiheaded attention layer. This will lead to overall better results as it was the main cause behind the state of the art results.



Update #3: Apply fusion layer

A fusion layer would be used after the feature extraction, whether by the convolution layers or the attention layers in order to combine the semantics extracted from the image to get better sense of the scenery and result in better overall colors.
